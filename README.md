# 🍎 AI Food Nutrition Analyzer

## 📌 Project Overview
AI Food Nutrition Analyzer is a machine learning-based web application that predicts the nutritional values of food items. Users can enter food-related information, and the application provides estimated nutrition details to help maintain a healthy diet.

## ✨ Features
- Predicts nutritional values using Machine Learning.
- User-friendly interface built with Streamlit.
- Fast and accurate predictions.
- Displays nutrition information clearly.
- Easy to use for anyone interested in healthy eating.

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Joblib

## 🚀 How to Run the Project

1. Clone the repository:
```bash
git clone https://github.com/artika-panwar/AI-FOOD-NUTRITION-ANALYZER.git
```

2. Install the required libraries:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run app.py
```

## 📊 Dataset
The model is trained on a food nutrition dataset containing different food items and their nutritional values.
